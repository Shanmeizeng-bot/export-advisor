# 出口目的国智能推荐系统 — 决策逻辑讲解

---

## 一、系统整体流程（一句话概括）

> 用户输入产品和企业信息 → AI自动匹配HS编码 → 从UN Comtrade实时抓取55国贸易数据 → 7维度算法打分 → 大模型生成个性化出口策略。

---

## 二、数据从哪里来？（RPA自动采集层）

我们不是用静态数据，而是每次分析时**实时从 UN Comtrade（联合国商品贸易统计数据库）抓取最新数据**。

| 数据类型 | 来源 | 具体内容 |
|---------|------|---------|
| 各国进口总额 | UN Comtrade API | 目标国当年从全球进口该HS编码产品的总金额 |
| 进口增速 | UN Comtrade API | 对比前一年的同比增长率 |
| 中国市场份额 | UN Comtrade API | 目标国从中国进口该产品的金额 / 该国总进口金额 |
| 国家画像 | 自建知识库 | GDP per capita、货币强度、物流LPI、电商渗透率、政策风险等 |
| FTA关税 | 自建知识库 | RCEP、中国-东盟FTA、中韩FTA的关税减让表和原产地规则 |

## 三、7维度评分算法（核心决策引擎）

总分100分，7个维度各自打分后相加，分数越高越推荐。

### 维度1：市场规模（0-20分）

**数据来源**：UN Comtrade 中目标国对该HS产品的进口总额。

**计算逻辑**：
- 采用**对数归一化**——因为不同国家的进口额可能差1000倍（如美国进口100亿美元 vs 肯尼亚进口100万美元），直接用线性归一化会让小市场永远得不到分
- 对数归一化后，差距被压缩到合理范围，小市场也能获得有意义的分值
- 再乘以**GDP乘数**做微调：同样的进口额，在高GDP国家意味着更高单价、更高利润

**举例**：塑料杯（HS 392410），美国进口$5亿 → 约18分；越南进口$800万 → 约4分。

### 维度2：市场增长（0-15分）

**数据来源**：UN Comtrade 同比增长率（当年 vs 前一年）。

**计算逻辑**：分段打分

| 增速 | 得分 | 含义 |
|------|------|------|
| >20% | 15分 | 爆发式增长，风口市场 |
| 10-20% | 12分 | 快速增长 |
| 5-10% | 9分 | 稳健增长 |
| 0-5% | 6分 | 微增 |
| -5-0% | 3分 | 轻微萎缩 |
| <-5% | 0分 | 明显萎缩 |

**设计理念**：增速反映的是"势头"——一个现在规模小但增速30%的市场，可能比一个规模大但增速1%的市场更有未来价值。

### 维度3：市场质量（0-20分）

**数据来源**：自建知识库（世界银行、IMF等权威来源）。

**计算逻辑**：三个子维度相加

| 子维度 | 分值 | 评估指标 |
|--------|------|---------|
| 消费能力 | 0-8分 | GDP per capita 对数归一化 |
| 货币强度 | 0-7分 | 是否为硬通货（USD/EUR/GBP等），汇率稳定性，结汇便利度 |
| 价格容忍度 | 0-5分 | 消费者是否愿意为品质付溢价（"极低"敏感度=5分，"极高"=0分） |

**设计理念**：市场规模大≠能赚钱。同样是进口1亿美元，在美国（人均GDP $85,000）意味着高单价高毛利，在印度（人均GDP $2,900）意味着薄利多销。这个维度专门衡量"赚不赚钱"。

**货币强度举例**：
- 美元/欧元：6分（全球储备货币，结汇零障碍）
- 日元/韩元/澳元：4分（发达国家货币，汇率稳定）
- 越南盾/印尼盾：1分（新兴市场货币，汇率波动大）

### 维度4：关税优势（0-15分）

**数据来源**：中国自由贸易区服务网（fta.mofcom.gov.cn）中各FTA协定文本的关税减让表。

**计算逻辑**：按FTA覆盖情况和最惠国税率综合评估

| 情况 | 得分 | 说明 |
|------|------|------|
| 双边FTA + RCEP双重覆盖 | 15分 | 如韩国：中韩FTA已零关税 + RCEP备选路径 |
| 中国-东盟FTA覆盖 | 13-14分 | 如越南/泰国：绝大多数产品零关税 |
| 仅RCEP覆盖 | 10-11分 | 如日本/澳大利亚：逐步降税中 |
| WTO最惠国税率≤3% | 9分 | 如新加坡：本身低关税 |
| WTO最惠国税率 3-8% | 6分 | 中等关税 |
| WTO最惠国税率 8-15% | 3分 | 较高关税 |
| WTO最惠国税率>15% | 1分 | 高关税壁垒 |

**设计理念**：FTA优于WTO最惠国待遇。同样的产品，走中韩FTA可能零关税，走WTO要交8%。关税差1%就是数万美元的利润差别。

**实际案例**：
- 塑料杯（HS 392410）出口韩国：中韩FTA零关税（15分）
- 塑料杯出口印度：WTO税率约15-20%（约2分）
- 这个差距直观反映了两国在关税维度上的天壤之别

### 维度5：竞争格局（0-12分）

**数据来源**：UN Comtrade 中"中国占目标国该产品进口总额的比重"。

**核心创新：区分市场类型打分**

传统做法是"中国份额越低=蓝海=分数越高"，但这忽略了市场差异：

| 市场类型 | 中国份额 | 得分 | 逻辑 |
|---------|---------|------|------|
| **高消费市场**（人均GDP≥$30,000） | <5% | 6分 | 还没进去，可能认证门槛高 |
| | 5-15% | 10分 | 有增长空间+已被验证可行 |
| | 15-35% | **12分** | 中国产品已有竞争力，最优区间 |
| | 35-55% | 9分 | 较成熟但仍有空间 |
| | >75% | 2分 | 红海，利润薄 |
| **新兴市场**（人均GDP<$30,000） | <15% | **12分** | 蓝海，先发优势 |
| | 15-30% | 9分 | 早期进入者 |
| | 30-50% | 6分 | 竞争加剧 |
| | >70% | 1分 | 红海 |

**设计理念**：在高消费力市场（如美国、德国），中国份额15-35%反而是好事——说明中国产品已经被市场接受，有竞争力，同时还有增长空间。而在低消费力市场（如柬埔寨），份额越低越好，因为壁垒低、可以先占。

### 维度6：合规与准入（0-10分）

**数据来源**：FTA协定文本中的原产地规则 + 自建知识库中的政策风险/贸易壁垒评估。

**计算逻辑**：合规容易度（0-5分）+ 市场准入难度（0-5分）

**合规容易度**：根据FTA原产地规则判定

| 难度 | 得分 | 典型情况 |
|------|------|---------|
| 低 | 5分 | 规则简单，如"品目改变"或低RVC门槛 |
| 中 | 3分 | 需要RVC 40%+品目改变双重条件 |
| 高 | 1.5分 | 严格的原产地审查 + 复杂文件要求 |

**市场准入难度**：根据政策风险和贸易壁垒综合评估

| 风险等级 | 含义 | 典型国家 |
|---------|------|---------|
| 极低 | 零关税自由港，几乎无障碍 | 新加坡 |
| 低 | 政策稳定，FTA覆盖好 | 韩国、澳大利亚、越南 |
| 中 | 有一定认证要求或政策不确定性 | 德国（CE认证）、法国 |
| 高 | 关税高 + 认证复杂 + 政策不确定 | 印度（BIS认证）、巴西 |
| 极高 | 制裁风险或重大贸易壁垒 | 俄罗斯 |

### 维度7：商业便利（0-8分）

**数据来源**：世界银行 LPI（物流绩效指数）、电商渗透率、营商环境报告。

**计算逻辑**：三个子维度相加

| 子维度 | 分值 | 数据来源 |
|--------|------|---------|
| 物流绩效 | 0-4分 | 世界银行 LPI（2.5-4.5范围映射） |
| 电商渗透 | 0-2分 | 各国电商渗透率×2 |
| 营商环境 | 0-2分 | 世界银行 Ease of Doing Business |

**设计理念**：产品好、关税低，但如果物流慢、清关难、电商不发达，中小企业依然做不起来。这个维度专门衡量"好不好做"。

## 四、FTA 如何影响推荐结果？

不同类型的贸易协定对推荐结果的影响路径：

### RCEP（区域全面经济伙伴关系协定）
- **覆盖国家**：日本、韩国、澳大利亚、新西兰 + 东盟10国
- **影响维度**：关税优势（+10-11分）、合规准入
- **关键条款**：允许**区域累积**——只要零部件来自RCEP成员国，都可以计入原产地价值成分
- **系统体现**：如果用户提到供应商在越南/泰国，系统会建议走RCEP路径利用累积规则

### 中国-东盟FTA（ACFTA）
- **覆盖国家**：越南、泰国、马来西亚、印尼、新加坡等
- **影响维度**：关税优势（+13-14分）、合规准入（规则相对简单）
- **系统体现**：东盟国家在关税维度得分普遍较高

### 中韩FTA（双边）
- **覆盖国家**：韩国
- **影响维度**：关税优势（+15分，最高）、合规准入
- **系统体现**：韩国在关税维度总是满分

### WTO最惠国待遇
- **覆盖国家**：所有WTO成员
- **影响维度**：关税优势（基准线，按实际税率倒扣）
- **系统体现**：没有FTA覆盖的国家（如印度、巴西），关税得分取决于其WTO承诺税率

## 五、AI大模型在做什么？

算法给出的是"分数"，但分数不能直接告诉企业"该怎么做"。

AI大模型（DeepSeek）接收的是**结构化上下文**：

1. 企业画像（名称、规模、出口经验、是否有电商团队）
2. 产品信息（HS编码、品类）
3. Top 5国家排名和各维度得分
4. Top 3国家的详细贸易数据
5. 行业知识库（相关展会、认证要求）

AI的输出是**5个板块**：

| 板块 | 内容 | AI的独特价值 |
|------|------|------------|
| 策略总览 | 2-3句话给出首选市场+核心理由 | 结合企业画像个性化（如"你有电商团队，建议直接做Amazon"） |
| 目标市场分析 | 每个市场的机会/认证/渠道/风险 | 结合具体展会名称、认证名称、平台名称 |
| 参展计划 | 3-5个最相关展会的表格 | 从知识库中检索并做推荐排序 |
| 行动时间线 | 准备期→切入期→扩张期的步骤 | 根据企业规模定制节奏（中小企业聚焦1-2个市场） |
| 关键提醒 | 1-2条容易被忽略的建议 | 如"你有东南亚出口经验，但韩国的认证要求完全不同" |

**关键设计原则**：AI不计算税率（避免幻觉），只做解读和策略生成。税率、贸易数据等"硬数据"完全来自结构化数据源。

## 六、一个完整案例：从输入到输出

以"东莞市卡珀包装科技有限公司，一次性塑料杯（HS 392410），PP材质"为例：

### Step 1：AI提取信息
- 企业名称：东莞市卡珀包装科技有限公司
- 产品：一次性塑料杯
- 材质：PP（聚丙烯）
- HS编码匹配：392410（塑料餐具及厨房用具）

### Step 2：RPA抓取数据
- 从UN Comtrade查询55个国家2023年HS 392410的进口数据
- 同时查询各国从中国的进口数据（计算中国市场份额）
- 耗时：约30-50秒（取决于查询国家数量）

### Step 3：7维度评分（以韩国为例）

| 维度 | 原始数据 | 计算过程 | 得分 |
|------|---------|---------|------|
| 市场规模 | 韩国进口$1.2亿 | 对数归一化 + GDP乘数 | 16/20 |
| 市场增长 | 同比增长8% | 8%归入"5-10%"档 | 9/15 |
| 市场质量 | 人均GDP $35,000 + 韩元 + 低敏感度 | GDP 6 + 货币 4 + 敏感度 4 | 14/20 |
| 关税优势 | 中韩FTA已零关税 | 双边FTA+RCEP双重覆盖 | 15/15 |
| 竞争格局 | 中国份额22% | 高消费市场，15-35%=最优区间 | 12/12 |
| 合规准入 | 低风险+低壁垒 | 合规4 + 准入5 | 9/10 |
| 商业便利 | LPI 3.8 + 电商92% + 高便利 | 物流3 + 电商2 + 营商2 | 7/8 |
| **总分** | | | **82/100** |

### Step 4：AI生成策略

基于82分排名第一的结果，AI会生成：

- **策略总览**：卡珀包装首选韩国市场——中韩FTA零关税+电商渗透率全球最高+Coupang平台适合中小企业起步
- **参展计划**：首尔国际食品包装展（具体时间+地点）
- **行动时间线**：1-3个月完成KCL认证→3-6个月上线Coupang→6-12个月拓展B2B渠道
- **关键提醒**：韩国食品接触材料需KFDC认证，与中国GB标准有差异

## 七、系统局限性（课堂坦诚说明）

1. **关税数据非实时**：FTA关税减让表来自PDF手动提取，可能存在版本滞后
2. **国家画像覆盖不全**：55个国家中，部分国家的画像数据（如电商渗透率）依赖估算
3. **HS编码匹配精度**：AI匹配6位HS编码准确率约85%，非常见产品可能需要手动输入
4. **不考虑非关税壁垒细节**：如特定行业的配额、反倾销调查等不在当前模型内
5. **企业规模适配简单**：目前仅区分"中小企业"和"大型企业"，未来可细化
